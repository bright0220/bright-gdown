=====
gdown
=====

Google Drive direct download of big files


Usage
=====

::

  gdown "[URL]" -O [FILENAME]

ex)

::

  gdown "https://drive.google.com/uc?id=0B9P1L--7Wd2vVlpzRkwzYUladms&export=download" -O hoge.txt


Installation
============

::

  pip install gdown
